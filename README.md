# 📚 PROA - Estudos de Desenvolvimento

Este repositório contém meus estudos e exercícios realizados durante minha formação no PROA (PROAProfissão).

Organizar minha evolução na área de desenvolvimento de software, registrando aprendizados, práticas e desafios resolvidos.

* Lógica de programação
* Java (Fundamentos e POO)

* React

## 🛠 Tecnologias

* Java
* MySQL
* HTML5
* CSS3
* JavaScript
* React
* Git e GitHub

## 📈 Organização

Este repositório está organizado por áreas de estudo.

Na pasta `web/`, trabalho com exercícios e aulas de desenvolvimento front-end usando **HTML e CSS**.

Na pasta `logica/`, vou concentrar exercícios, desafios e anotações de raciocínio lógico usando **JavaScript**.

Na pasta `react/`, vou organizar as tarefas e práticas de **React**.
